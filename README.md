# nodesk

Notion'a kaydeden, Apple Notes estetiğinde, masaüstü sürüklenebilir widget'lı not uygulaması.
Tauri 2 + React + TipTap + OpenRouter AI.

## Hızlı başlangıç
```bash
cp .env.example .env
# .env içine NOTION_API_KEY, NOTION_DATABASE_ID, OPENROUTER_API_KEY doldur

npm install
npm run tauri dev
```

Detaylar için [CLAUDE.md](./CLAUDE.md).
